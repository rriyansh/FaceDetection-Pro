# ReyProject 🚀

Control your Mac with just your **hands**, **mouth**, and **eyes** – plus **air drawing**!

##  Features

-  **Both hands visible** → Opens Terminal  
-  **No hands visible** → Closes Terminal  
-  **Mouth open** → Opens Safari  
-  **Mouth closed** → Closes Safari  
-  **Eyes closed for 5 seconds** → Beep alert (prevents dozing off)  
-  **Index finger up** → Draw on screen  
-  **Index + middle fingers up** → Erase  
-  **Press 'c'** → Clear entire drawing  
-  **Press 'q'** → Quit

All processing is **offline**, real-time at **60 FPS** on a standard webcam.

## 🖥️ Requirements

- Python 3.8+
- macOS (Linux/Windows support for terminal commands may differ)
- Webcam
